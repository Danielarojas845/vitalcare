##Clínica VitalCare
Entrega del mòdulo 7, enfocado en la creación de un sistema web utilizando Django, con el objetivo de aplicar y demostrar conceptos fundamentales de desarrollo backend y estructura de aplicaciones web.


##Descripción del Proyecto
Clínica VitalCare es una aplicación web básica que simula la gestión de una clínica, permitiendo trabajar con estructuras típicas del desarrollo web como rutas, vistas, modelos y plantillas.


##Funcionalidades principales
Estructura base de un proyecto Django
Separación por aplicaciones internas
Uso de vistas, modelos y plantillas HTML
Gestión de doctores
Gestión de horas de atención
Integración con base de datos PostgreSQL
Acceso al panel de administración de Django


##Accesos principales del proyecto
Una vez ejecutado el proyecto en entorno local, se puede acceder a las siguientes rutas:


##Página de inicio
http://127.0.0.1:8000/

##Listado y gestión de doctores
http://127.0.0.1:8000/doctores/

##Gestión de horas médicas
http://127.0.0.1:8000/horas/

##Panel de administración
http://127.0.0.1:8000/admin/


El panel de administración permite gestionar los registros del sistema y visualizar el funcionamiento interno de la aplicación.
